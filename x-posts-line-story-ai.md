# line-story-ai X投稿ドラフト

## パターン1: Before/After披露型(最優先)

```
> LUNA_AI.exe — デモログ

文章を1つ入力するだけで、LINEのトーク画面風アニメーションが完成する。

「すみません」じゃなくて「おはようございます」だろ
→ お、おはようございます!

これだけのやり取りが、8秒で動画になる。

#個人開発 #AI #LINEトーク
```

## パターン2: 相談に応えた型

```
> LUNA_AI.exe — つぶやき

「あの修羅場LINE、動画にして見せたい」って相談、結構もらうんだ。

文章を箇条書きで入れるだけで、AIが自然な会話に分解して、LINEトーク風アニメーションに変換してくれる。

AI音声つきで保存もできるようになった。スマホでも動く。

良ければ使ってみて。

#個人開発 #AI開発
```

## 補足

- いずれもURLはプロフィール欄(ai-builder-site)への誘導を想定
- スクショ/動画添付案: 実際に生成したLINEトーク風アニメーションのスクリーン録画が一番効果的。個人情報が映る入力文章は避け、デモ用の無難な文章(待ち合わせ・雑談など)で生成すること
- 投稿前にdemo動画を1本作っておく(このまま貼れる素材として保存)
